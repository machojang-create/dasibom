# 배포 따라하기 (프로그래머 아니어도 OK)

> 목표: 지금까지 만든 화면을 실제 사이트에 올리기(=배포).
> 컴퓨터에 도구가 아직 없어서 **설치(1~6단계)** 부터 합니다. 한 번만 설치하면 다음부턴 5·6·7단계만 하면 됩니다.
> 막히면 **빨간 글씨(ERROR) 줄을 캡처해서** 보내주세요.

---

## 1단계. Node.js 설치 (firebase 도구를 쓰기 위한 기반)
1. 브라우저에서 **https://nodejs.org** 접속
2. 초록 버튼 두 개 중 **왼쪽 "LTS"** 클릭 → 설치파일(.msi)이 다운로드됨
3. 다운로드된 파일 **더블클릭** → 설치창이 뜨면:
   Next → "I accept..." 체크 → Next → Next → Next → **Install** → (잠깐 기다림) → **Finish**
   - 중간에 "Tools for Native Modules" 같은 체크박스는 **안 건드려도** 됩니다.

## 2단계. PowerShell(파란 명령창) 열기
1. 키보드 **윈도우키**(⊞) 누르기
2. **powershell** 이라고 타이핑
3. **"Windows PowerShell"** 클릭 → 파란/검은 창이 열림
   - 관리자 권한 필요 없음. (Node 설치 *후에* 열어야 인식됩니다)

## 3단계. firebase 도구 설치
파란 창에 아래 한 줄을 **복사 → 붙여넣기**(마우스 우클릭 또는 Ctrl+V) → **Enter**:
```
npm install -g firebase-tools
```
- 1~3분 걸립니다. 글자가 주르륵 지나가다 멈추면 완료.
- **노란색 WARN(경고)** 는 무시해도 됩니다. **빨간 ERROR** 만 문제예요.
- 확인: 아래 입력 → Enter → 숫자(예: 13.x.x)가 나오면 성공
```
firebase --version
```
> ⚠️ 만약 빨간 글씨로 **"...실행할 수 없습니다 / running scripts is disabled..."** 가 뜨면,
> 아래 한 줄을 입력 → Enter → 묻는 말에 **Y** → Enter 하고 다시 `firebase --version`:
> ```
> Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
> ```

## 4단계. firebase 로그인 (한 번만)
```
firebase login
```
1. "Allow Firebase to collect... ?" 물으면 → **Y** 입력 → Enter (아무거나 괜찮음)
2. **브라우저가 자동으로 열림** → 다시봄 프로젝트를 만든 구글계정(**machojang@gmail.com**) 선택 → **"허용(Allow)"** 클릭
3. 브라우저에 **"Firebase CLI Login Successful"** 뜨면 그 탭 닫기
4. 파란 창에 **"✔ Success! Logged in as machojang@gmail.com"** 비슷한 줄이 나오면 완료

## 5단계. 프로젝트 폴더로 이동
아래 한 줄 복붙 → Enter:
```
cd "C:\Users\USER\Desktop\이전작업\백업\memoir"
```
- 줄 맨 앞 경로가 ...\memoir> 로 바뀌면 OK.

## 6단계. ★배포! (화면만 올리기)
아래 한 줄 복붙 → Enter:
```
firebase deploy --only hosting
```
- 1~2분 후 초록 글씨로:
  ```
  ✔  Deploy complete!
  Hosting URL: https://mylife-650f0.web.app
  ```
- 그 **Hosting URL** 을 브라우저에서 열어 화면 확인. 끝!

## 7단계 (선택). 공유링크 규칙도 올리기
자서전 "공유 링크(3일)" 를 실제로 쓰려면:
```
firebase deploy --only firestore:rules
```

---

## ❌ 절대 입력하면 안 되는 것
```
firebase deploy
```
→ `--only` 없이 이것만 치면 **functions(서버)까지 배포**되어, 유료 Opus가 결제검증 없이 켜집니다(비용 위험).
**꼭 `firebase deploy --only hosting` 처럼 `--only` 를 붙이세요.**
