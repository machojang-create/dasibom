# 그때그시절 이미지·설명 자동 생성 (제미나이 나노바나나)

> 300개 추억(과거+현재 = 최대 600장)의 그림과 설명을 자동 생성합니다.
> 프롬프트는 데이터에 이미 다 들어있어서, **키만 있으면 명령 한 줄**입니다.

## 1. 제미나이 API 키 (1회, 무료)
1. https://aistudio.google.com/apikey 접속 (구글 로그인)
2. **"API 키 만들기"** 클릭 → 생성된 키 복사 (⚠️ 채팅·메일에 붙이지 말 것)

## 2. 파란 창(PowerShell)에서
```
cd "C:\Users\USER\Desktop\이전작업\백업\memoir"
$env:GEMINI_API_KEY="여기에키붙여넣기"
```

**이미지 생성 — 우선 30개(60장)로 시작 권장:**
```
node tools/gen_nostalgia.mjs images --from 1 --to 30
```
- 장당 ~7초 간격(무료 한도 보호) → 60장 ≈ 7분. 이미 만든 건 건너뛰니 **끊겨도 같은 명령으로 이어돌리기** 됩니다.
- ⚠️ 무료 키는 일일 한도가 있어요(이미지 모델은 보수적으로 잡으면 하루 수십~백 장). 한도 걸리면 내일 같은 명령. 유료 결제 연결 시 장당 약 $0.04 (600장 ≈ $24).

**설명 보강 + 말벗 질문 (텍스트, 한도 넉넉):**
```
node tools/gen_nostalgia.mjs desc --from 1 --to 300
```

**추억 퀴즈 300문제 생성 (텍스트, 한도 넉넉):**
```
node tools/gen_nostalgia.mjs quiz --from 1 --to 300
```
- 매일 1문제 "오늘의 추억 퀴즈"(도장판·추억첩)용. 약 12분.
- 과거/현재 설명을 2문장으로 풍부하게 + 🗣️ "말벗 한마디"(가족과 대화 유도 질문) 추가
- 원본은 nostalgia_data.json.bak 으로 자동 백업

## 3. 결과 반영 (생성 후)
생성된 그림 몇 장을 폴더(nostalgia/img)에서 열어 마음에 드는지 확인하시고:
```
git add -A
git commit -m "그때그시절 이미지·설명 생성"
git push
```
→ 1분 뒤 사이트 자동 반영. (또는 저한테 "생성 끝났어"라고 하시면 확인 후 제가 마무리)

## 메모
- 그림 스타일: 따뜻한 파스텔 플랫 일러스트(프롬프트에 지정돼 있음) + 글자 없음 가드 자동 추가
- 마음에 안 드는 장은 해당 파일만 지우고 같은 명령 → 그 장만 다시 생성
